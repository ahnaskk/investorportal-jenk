User Tpes Provided In The Investor Portal

Investor Portal provides us with various user types. Each user is characterized with the functionalities provided to them by the Admin, according to the principles of the Merchant Cash Advance(MCA) Industry.

The different types of users are as given below:

  • Admin User Type
  • Companies User Type
  • Merchant User Type
  • Investor User Type
  • Lender User Type
  • Editor User Type
  • Viewer User Type
Admin

Admin has been given the highest priority among all users. He has been allowed to Create, View and Edit all other user transactions and almost able to perform all important functions provided in the Investor Portal software.

Companies

Companies are the main players in the Investor Portal software scenario. At present, we have two main companies in our software. They are VP Advance Funding and Velocity.

Merchants

Merchants are the borrowers in the MCA industry. They submit their requests to the Lenders who in turn forward them to the Companies. Once request are passed, they are released to the Merchant through the Lenders.

Investors

Investors invest their cash in the companies. These finances are directed towards the MCA business.

Lenders

Lenders act as intermediator between the Companies and the Merchants. They bring in the Merchants to the Company. They put forward their requests and once funds are released, these are directed to Merchants. Even the cash payback process goes through the Lenders. They get paid with commission for the same.

Editors

Another type of a users in Investor Portal are the Editors. They have the permission of viewing with Admin permissions. They are also permitted to edit payments done.

Viewers

Viewers are also allowed to only view and check Investor Portal with Admin permissions.
